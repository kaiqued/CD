Case (debate) sobre Big Data

Big Data é o armazenamento de tudo aquilo o que o sistema (a rede) é capaz de reunir. A partir dessa afirmação, o grupo discorreu sobre a importância e a influência do Big Data no mundo atual, bem como o crescimento de tal importância/influência, exemplificando as utilidades de toda essa informação para o desenvolvimento de novas tecnologias, por exemplo. O homem produz dados e, a interpretação desses dados os levam a novas criações. Novas criações os levam a maior produção de dados e, esse ciclo se repete indefinidamente. Big Data também é tema trabalhado na Literatura, sendo exemplos de obras relacionadas a esse tema também apresentados pelos alunos.

Ideias de Altruísmo Efetivo aplicadas a engenharia - Artigo científico

Este artigo apresentra e debate as ideias de altruísmo efetivo de Peter Singer para uma se encaixar a visão que o engenheiro deve ter da sociedade trabalhada ao longo do curso. Discutir assuntos como qual deve ser a prioridade de qualquer indivíduo capaz de mudar o mundo e que métodos utilizar para faze-lo.
